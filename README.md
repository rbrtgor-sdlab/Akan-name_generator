# Akan Name Generator
# github link

## Description

Akan Name Generator is a web application that determines your Akan name based on the day of the week you were born. In African tradition, particularly among the Akan people of Ghana, children are given names based on the day of the week of their birth. This application helps you discover your traditional Akan name.

## Features

- **Date Input**: Enter your date of birth to calculate the day of the week
- **Gender Selection**: Choose your gender (Male or Female) to get the appropriate Akan name
- **Instant Results**: Display your birth day and corresponding Akan name
- **User-Friendly Interface**: Clean and intuitive web interface

## How It Works

1. Enter your date of birth in the date input field
2. Select your gender from the dropdown menu
3. Click the "Submit" button
4. Your Akan name and birth day will be displayed instantly

## Akan Names by Day

### Male Names
- Sunday: Kwasi
- Monday: Kwadwo
- Tuesday: Kwabena
- Wednesday: Kwaku
- Thursday: Yaw
- Friday: Kofi
- Saturday: Kwame

### Female Names
- Sunday: Akosua
- Monday: Adwoa
- Tuesday: Abenaa
- Wednesday: Akua
- Thursday: Yaa
- Friday: Afua
- Saturday: Ama

## Installation & Usage

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Fill in your date of birth and select your gender
4. Click Submit to generate your Akan name

No dependencies or installation required - this is a standalone web application.
